# Designing Agentic Workflows: Hands on Workshop

Cloudera AI Studios is a suite of low-code tools designed to streamline the development and deployment of generative AI solutions within enterprises.

This hands-on workshop, "Designing Agentic Workflows," focuses on teaching participants how to design and build end-to-end agentic workflows using **Cloudera AI Agent Studios**. The theme centers around applying agentic workflow design to a real-world use case, allowing participants to create efficient and intelligent solutions with ease.

!!! info "Use Case and Objective"
    Your objective in this workshop is to address this issue and build an Air Quality Investigator System to provide accurate Air Quality Analysis.
